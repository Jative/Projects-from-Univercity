Разработка электронных таблиц.

Общие требования к пользовательскому интерфейсу:
1.  На  экране  появляется  на  20  секунд  форма-заставка  с  произвольным графическим изображением и текстовой информацией о данном приложении.
Далее  запускается  приложение  с  возможностью  создания,  открытия, изменения  и  сохранения  файла  по  типу  электронной  таблицы (многодокументный интерфейс).

2. Приложение должно содержать:
Меню приложения;
Работа с файлами;
создание файла собственного формата;
открытие файла собственного формата;
закрытие файла собственного формата;
сохранение файла собственного формата;
сохранение таблицы в файл формата Excel;
сохранение таблицы в файл формата Word. 


Для  выбора,  открытия  или  сохранения  файла  использовать  стандартные диалоги OpenFileDialog и SaveFileDialog. При  закрытии  дочернего  окна  в  котором  были  внесены  изменения, программа должна предлагать сохранить эти изменения.
Если файл новый, то сохранить с возможностью задания имени файла.

Окно:
расположение окон каскадом;
расположение окон вертикально;
расположение окон горизонтально;
перечень открытых дочерних окон.

Задание:(согласно варианта).
Панель  инструментов с  быстрыми  кнопками,  каждая  кнопка  должна содержать соответствующую подсказку согласно производимой операции.
Кнопки должны дублировать системное меню.
Контекстное меню для работы ячейками таблицы.

3. Предусмотреть механизм ввода, изменения и удаления информации из ячеек, возможность распознавания текстовой и числовой информации.

4. Создание диаграммы по выделенному диапазону. К одной дочерней форме можно создавать несколько диаграмм.
При изменении значений в ячейках,относящихся к диапазону на основе которого построена диаграмма, должна изменятся соответствующая диаграмма.
Диаграммы должны быть связанны с дочерней  формой.
При  закрытии  дочерней  формы,  связанные  формы-диаграммы должны закрываться.

Для числовой информации: возможность подсчета количества четных чисел в выделенных ячейках.
В случае невозможности выполнения выдать соответствующее сообщение.

Для текстовой информации: занесение в ячейку строки получаемой в результате выбора  из  выделенных  ячеек  всех  слов,  в  которых  есть словосочетание "Del".
Вслучае невозможности найти такие слова выдать соответствующее сообщение.
