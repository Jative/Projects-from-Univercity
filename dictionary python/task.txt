Используя  тип  словарь  из  пар термин:толкование_термина,  создать программу для выполнения основных действий с глоссарием терминов. Требования к программе:

1. В программе должно быть реализовано меню, содержащее следующие пункты:
а) Создание глоссария(на основе текстового файла)
б) Дополнение глоссария парами термин:тол  кование_термина.
в) Удаление из глоссария пар термин:толкование_термина.
г) Вывод глоссария на экран.
д) Поиск в глоссарии толкования термина и вывод его на экран.
е) Сохранение глоссария в фале.
з) Загрузка глоссария из файла.
ж) Все задания должны быть оформлены отдельной функцией.

2.  Глоссарий  должен  содержать  толкование  терминов по  следующей тематике: Зарезервированные и ключевые слова C++.

Входные значения: Целое число command, определяющее то, что будет делать код. Дальше в зависимости от того, что будет запрашивать программа.
Выходные значения: В зависимости от режима то, что запрашивает пользователь: толкование конкретного термина, весь глоссарий и т.д.
