Базовый уровень
а) Реализовать класс «Точка», содержащий в качестве полей название точки (А, В, С и т.д.) и значение координат точки (X, Y).
В классе предусмотреть:
•конструкторы по умолчанию и конструкторы с параметрами;
•свойства.
б) Реализовать  класс  геометрическая  фигура  на  плоскости  (Круг).
В классе предусмотреть:
•поле – массив класса «Точка»;
•индексаторы, свойства;
•конструкторы;
•свойство: вычисления площади фигуры(только для чтения);
•событие  (площадь  фигуры  равна  1).
Обработчик  события  должен  выводить  старое  и новое  значение  площади  на  экран(старое  значение  площади  передавать  в  качестве параметра события);
•предусмотреть  возможность  обработки  исключений  при  вводе  данных  (стандартные виды исключений).

Повышенный уровень
В дополнение к базовому уровню реализовать свой класс исключение, основанный на классе Exception, которое возникает при невозможности построения данной фигуры исходя из введенных координат вершин фигуры.
Реализовать реакцию на данное исключение на основе события  и  обработчик  этого  события.  В  качестве  параметров  передавать  объект  класса «Точка» вызвавший исключение.
