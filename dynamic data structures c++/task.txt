1.Создать запись для хранения следующей информации:
наименование товара,
место покупки,
цена,
дата покупки.
2. Предусмотреть возможность добавления, изменения и удаление записи и отображения данных на экран. Для хранения данных использовать очередь.
3. Получить список мест покупки и наименований товаров, приобретенных во втором квартале позапрошлого года.
4. Найти среднюю стоимость покупок, сделанных по каждому из встречающихся мест покупки. Для группировки данных использовать стек.
5. Найти количество покупок за каждый месяц прошлого года.
Входные значения: заполненная пользователем очередь.
Выходные значения: вывод очередей на экран; места покупки и наименования товаров второго квартала позапрошлого года; средняя стоимость покупок, по каждому месту покупки; количество покупок за каждый месяц прошлого года.
