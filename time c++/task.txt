Описать структуру «время» (часы, минуты, секунды).  Определить функцию «прошедшее время» определяющую интервал времени между t1и t2 в секундах.
	Требования к программе:
1. В программе должно быть реализовано меню, содержащее следующие пункты:
а) Добавление структуры в массив.
б) Редактирование структуры.
в) Удаление структуры из массива.
г) Вывод элементов массива на экран.
д) Вызов функций согласно заданию(предусмотреть возможность выбора пользователем данных из массива для передачи в функцию).
2. Все задания должны быть реализованы в виде функций.
3. Функции должны иметь независимый интерфейс.
4. Все функции, используемые в проекте должны содержаться в отдельном модуле.

Входные значения: целое число command, определяющее дальнейшую работу программы. Далее, в зависимости от комманды, вводить целые числа, определяющие индекс, часы, минуты и секунды.
Выходные значения: То, что запросит пользователь (Список всех сохранённых времён, разница во времени между двумя структурами по заданию).
