В строку S добавить необходимое количество пробелов так, чтобы ее длина стала равна n.
Причем: перед первым словом пробелы не добавлять, после  последнего  слова  все  пробелы  удалить,  добавленные  пробелы равномерно распределить между словами.
Если длина S превосходит n, удалить S из все слова, которые не укладываются в первые n символов, а оставшуюся часть преобразовать по вышеуказанным правилам.
Реализовать на языке C++, применяя массив символов.
Входные значения: Целое число n – конечная длина массива символов.
Выходные значения: Массив символов S, преобразованный согласно условиям.
